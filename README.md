# 🌐 PolyGlot AI - Enterprise Language Translation Engine

A high-performance, responsive language translation web application built as part of the **CodeAlpha Internship Program**. This project satisfies all core and optional requirements for **Task 1: Language Translation Tool** with production-ready UI/UX standards.

---

## ✨ Core Features Implemented

* **Dynamic User Interface**: Designed a fluid, dual-panel responsive workspace using Tailwind CSS and Lucide Icons for high scannability.
* **API Integration (Task 1)**: Integrated the asynchronous MyMemory Translation API to process inputs securely and fetch fast, accurate target text without brittle API key setups.
* **Text-to-Speech Engine (Optional Feature)**: Native browser voice synthesis wrapper integrated into both source and target sides for real-time pronunciation support.
* **Asynchronous Clipboard Controller (Optional Feature)**: A one-click smart copy button with dynamic state feedback (`✓ Copied!`).
* **Live Analytics State**: Real-time interactive character counters ($0 \text{ to } 5000$ characters) alongside custom loading spinners during network requests to prevent UI freezing.

---

## 🛠️ Tech Stack & Architecture

* **Frontend Layout**: HTML5 (Semantic Structure)
* **Styling Framework**: Tailwind CSS v3 (Utility-first configuration via CDN)
* **Icon Set**: Lucide Icons Layer
* **Logic Engine**: Vanilla JavaScript (ES6+ Asynchronous Fetch Architecture)
